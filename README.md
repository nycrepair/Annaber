# Портфолио — Киселева Анна Петровна

Профессиональный сайт-портфолио для SMM / PR / Контент-специалиста.

## Структура

- `index.html` — единственный файл сайта, не требует сборки

## Технологии

- Семантический HTML5 с атрибутами доступности (`aria-*`)
- CSS3: custom properties, анимации, адаптивный grid/flexbox
- Vanilla JS: Canvas-частицы, Intersection Observer, мобильное меню
- Google Fonts: Cormorant Garamond + Inter

## SEO

- `<title>`, `<meta description>`, Open Graph, Twitter Card
- Семантические теги: `<section>`, `<article>`, `<header>`, `<nav>`, `<footer>`
- `lang="ru"`, canonical URL
